# 🎬 VideoTube (Stream SaaS)

A full-stack video streaming platform built using the MERN stack.
It allows users to upload, manage, and analyze video content with a clean dashboard interface.

---

## 📸 Preview

![App Preview](./1773342613644.jfif)

---

## 🚀 Features

* 🔐 User Authentication & Authorization
* 🎥 Video Upload & Management
* 📊 Dashboard with Analytics (views, likes, subscribers)
* 📈 Visual Data Representation (charts)
* 🔎 Video Search Functionality
* 👍 Like & Subscribe System
* 🧑‍💻 User-specific content management

---

## 🛠 Tech Stack

**Frontend**

* React.js
* Tailwind CSS
* Axios

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Other Tools**

* JWT Authentication
* Multer (file uploads)

---

## 🧠 How It Works

* Users can register/login securely
* Upload videos with metadata
* View analytics like total views, likes, subscribers
* Dashboard displays insights using charts
* Users can interact with videos (like/subscribe)

---

## ⚙️ Installation

```bash
git clone https://github.com/surajnimbalkar07/VideoTube.git
cd VideoTube
npm install
npm run dev
```
